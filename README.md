## ✨ **Key Features**

| Section | Description |
|---------|-------------|
| 📊 **Dashboard** | Real‑time CPU (EWMA model), RAM, and queue length monitoring. |
| ⚖️ **Load Balancer** | Round Robin, Least Connections, Weighted, and Random algorithms. |
| 🌐 **Network** | Simulates RTT, jitter, and packet loss. |
| 📈 **Auto‑scaling** | Dynamic server scaling based on CPU threshold and queue size. |
| 🌍 **Bilingual** | Full English / Arabic support with RTL/LTR toggle. |

---

## 📖 **Project Concept & Security Focus**

This interactive simulator was developed to study **Queueing Theory** and **System Performance**. From a security perspective, it helps in understanding:
- How systems behave under **Stress Tests** and **DDoS simulation**.
- The impact of network latency and packet loss on service availability.
- Effectiveness of auto-scaling in mitigating resource exhaustion attacks.
